# Описание проекта

Репозиторий состоит из двух папок:
* `ClinicVS22` — решение Visual Studio (C#);
* `react-vite-intro` — фронтенд-приложение на React.

## Настройка и конфигурация

1. **База данных**: Укажите строку подключения `SqlConnection` в файле `appsettings.json`. Если вы используете не MS SQL, внесите соответствующие изменения в класс `InfrastructureConfigurator`.
2. **CORS**: В файле `Program.cs` настройте параметр `builder.WithOrigins`, указав адрес и порт вашего хоста для корректного взаимодействия с React-приложением.

## Описание программы

Разработана база данных со следующими сущностями:
* **Patients** (Пациенты);
* **Doctors** (Врачи);
* **Diseases** (Заболевания).

**Реализованный функционал:**
* Контроллеры для вывода информации как по конкретному пациенту, так и по всем пациентам сразу.
* Получение данных о врачах, сгруппированных по специальностям.
* Общий список заболеваний и контроллер для обновления данных.
* Простая страница на **React** для визуализации и получения данных.
